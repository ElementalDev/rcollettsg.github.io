Projects

Tutor Student Matching System

This is my final college project, written in C#. The user would be given a log in or has been made one (only the admins can make one) and then they would need to fill out a form to give their personal information, days that they were free and their learning style. An automated class then went through an Access Database to find a tutor with the same or close to the same schedule and the same learning style.

The Cave

This project is still a work in progress. The game involves the player choosing a profession at the beginning of the game. All the professions, weapons, armours and enemies have an is-a relationship with classes of the same name to prevent having to rewrite code that is common. They then need to choose a difficulty, which will determine how many moves they have to escape the cave. They then have to escape by going either forward, back, left or right. A RND determines when enemies spawn and when helpful items spawn so the user cant determine when something is going to happen. Once they escape, they can have one of 3 different endings, depending on how they got out, what they collected and their health at the end. This was done using several classes and procedures as well as concepts such as encapsulation and inheritance.
